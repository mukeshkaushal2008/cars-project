B) Customization - top

You can custom pretty much everything in the template - developer expecting skilled customers or co-developers.

File Structure

HTML

In the root folder you will find the core html file of the landing page. Here is where you can edit the actual contet of you site.

CSS

The css files located in the CSS directory are responsible for the look of your theme - here you can control the typography, paddings and positions of elements and colors.

CSS Files included:

  • style.css - this is the master styles file. Here you can edit every section of the site - notice that the file is actually structured so you can quickly find the section you want to edit
  • style980.css - this additional grid style to build default responsive grid for screen dimension from 800 to 980 pixels width (grid system 960)
  • style800.css - additional responsive grid for screen between 700 and 800 pixels width.
  • style700.css - additional responsive grid for screen between 600 and 700 pixels width.
  • style600.css - additional responsive grid for screen between 500 and 600 pixels width.
  • style500.css - additional responsive grid for screen between 400 and 500 pixels width.
  • style400.css - additional responsive grid for screen between 300 and 400 pixels width (handheld).
  • style_ie.css - additional CSS for IE customisation.
  • jquery.fancybox-1.3.4.css - fancybox stylesheet.
Javascript

The css files located in the js directory are responsible for javascript effects and functions.

JavaSripts Files included:

  • jquery-1.8.3.min.js - this is the jQuery javascript library (you don't need to edit it).
  • jquery.bxslider.js - this is the jQuery BxSlider v4.1 javascript library (you don't need to edit it).
  • jquery.bxslider.min.js - this is minified jQuery BxSlider v4.1 javascript library (you don't need to edit it).
  • jquery.carousel.js - this is the jQuery Carousel Plugin v1.0 javascript library (you don't need to edit it).
  • jquery.checkbox.js - Check Box customisation scripts.
  • jquery.countdown.js - Countdown for jQuery v1.6.2 library for underconstruction page.
  • jquery.easing.1.3.js - this is the jQuery Easing v1.3 javascript library (you don't need to edit it).
  • jquery.elastislide.js - this is touch devices library (you don't need to edit it).
  • jquery.fancybox-1.3.4.pack.js - this is FancyBox - jQuery Plugin library (you don't need to edit it).
  • jquery.mousewheel-3.0.4.pack.js - this is minified MouseWheel jQuery library by Brandon Aaron (you don't need to edit it).
  • jquery.mousewheel.js - this is MouseWheel jQuery library by Brandon Aaron (you don't need to edit it).
  • jquery.selectik.js - this is custom select dropdown customisation library.
  • jquery_old.js
  • js.js - this is the primary script where custom functions and plugin configurations.
  • mobilyslider.js - this is MobileSlider jQuery library (you don't need to edit it).

Editing

The HTML is well structured with comments on every section you need to edit.

Fonts

The fonts used are from the Google Web font library which also located inside fonts directory. You can change the font by editing the url provided in the HTML where marked and reflect that change in the css. To learn more and choose the perfect font for you site just head over to Google WebFonts - http://www.google.com/fonts/
